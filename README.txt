== Description ==

Gift Recorder will add a spiritual test page that users can use as a guest or as a signed in user of which will create a record of user data. Gift Recorder is a part of a Larger suite called Church Recorder that will facilitate a larger record of user accounts. The data that is kept can be utilized for analytics and even to provide member ministry information and schedules. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `gift-recorder.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('GIFT_RECORDER_hook'); ?>` in your templates
